# 🧭 CoinCompass — Умный финансовый трекер

**CoinCompass** — это веб-приложение для контроля личных финансов, которое объединяет простоту интерфейса с мощными облачными технологиями. Проект разработан в рамках хакатона для демонстрации навыков работы с динамическими данными и визуализацией.

### 🛠 Технологический стек
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Database:** Firebase Realtime Database — для мгновенной синхронизации данных.
* **Charts:** Chart.js — для построения интерактивных круговых диаграмм.
* **Hosting:** GitHub Pages.

### ✨ Ключевые фичи
1.  **Облачное хранение:** Ваши данные не пропадут при перезагрузке страницы. Они привязаны к вашему профилю в реальном времени.
2.  **Интерактивная аналитика:** Динамический график, который автоматически пересчитывает доли категорий при добавлении новых трат.
3.  **Адаптивный дизайн:** Удобно пользоваться как с компьютера, так и со смартфона.
4.  **Экспорт данных:** Возможность скачать всю историю операций в формате JSON.

### 📈 Как это работает
Приложение отправляет каждый введенный расход в облако Google Firebase. Скрипт подписывается на обновления базы данных: как только данные меняются, интерфейс и графики перерисовываются без перезагрузки страницы.
