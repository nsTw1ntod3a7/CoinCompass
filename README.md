# 🧭 CoinCompass

Персональный финансовый трекер с облачной синхронизацией.

### 🛠 Стек технологий:
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Backend:** Firebase Realtime Database.
* **Библиотеки:** Chart.js (визуализация данных).

### ✨ Основные функции:
* Авторизация пользователей.
* Добавление, редактирование и удаление расходов.
* Интерактивная диаграмма трат.
* Фильтрация статистики по месяцам и категориям.
* Экспорт данных в JSON.
